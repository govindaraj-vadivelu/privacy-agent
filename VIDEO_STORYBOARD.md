- [ ] Hide bookmarks bar
- [ ] Set browser to 1920x1080
- [ ] Test screen recorder (OBS, Loom, QuickTime)
- [ ] Close notifications
- [ ] Prepare demo data (service names ready)

### During Recording
- [ ] Smooth mouse movements (not too fast)
- [ ] Pause briefly before clicking
- [ ] Show key sections clearly

## ðŸ› ï¸ Editing Workflow

### Step 1: Rough Cut (Day 1)
1. Import all screen recordings
2. Arrange in sequence
3. Trim dead time
4. Check timing (2:00 total)

### Step 2: Add Voiceover (Day 1)
1. Record narration following script
2. Sync audio to video
3. Adjust pacing

### Step 3: Visual Polish (Day 2)
1. Add text overlays
2. Add transitions (simple fades)
3. Color correction (if needed)

### Step 4: Sound Design (Day 2)
1. Add background music
2. Balance audio levels (voice > music)
3. Add subtle sound effects (optional)

### Step 5: Export & Review (Day 2)
1. Export 1080p MP4
2. Watch full video
3. Get feedback
4. Make final adjustments
