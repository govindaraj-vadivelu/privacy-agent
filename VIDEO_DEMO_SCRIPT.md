## üìä Production Notes

### Technical Setup

**Tools needed:**
- Screen recording: OBS Studio, Loom, or QuickTime
- Video editing: DaVinci Resolve (free), iMovie, or Descript
- Voiceover: Your voice + Audacity (free) or ElevenLabs TTS
- Music: Epidemic Sound, Artlist, or YouTube Audio Library

### Recording Steps

1. **Record screen captures:**
   - Clean browser window (no distractions)
   - Start frontend: `npm run dev`
   - Record full demo flow
   - Capture key moments: input ‚Üí processing ‚Üí results

2. **Record voiceover:**
   - Use script above
   - Clear, confident tone
   - Slightly fast pace (keeps energy up)

3. **Edit video:**
   - Cut dead time
   - Add on-screen text overlays
   - Add background music (subtle, not distracting)
   - Add transitions between sections

4. **Export:**
   - 1080p (1920x1080)
   - 30 fps
   - MP4 format
   - Under 50 MB for easy sharing

### Visual Style

- **Color scheme:** Clean, modern, privacy-focused
- **Font:** Sans-serif, high contrast
- **Animations:** Smooth, minimal
- **Music:** Upbeat but not overwhelming (tech/startup vibe)
